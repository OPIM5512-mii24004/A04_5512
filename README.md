# A04_5512
This repository contains the code for assignment A04. Here, loan data was used to build a classification model, investigate the most important predictive features, and assess the relationship between the features and the predicted outcome (loan status). To run this code, install all required packages and import the loan data. The script will start with cleaning the data and label encoding the target variable. The data is then split into training and testing sets, categorical variables are label-encoded, and preprocessing pipelines are generated for our potential models of interest. Grid search parameters are set, and the grid search is conducted to identify the parameters that yield the best classification model. With the best model, permutation importance is performed to determine the top 5 predictive features, which are further examined with an ICE plot and PDP.
